# CV Project Bullets

## 大模型 RAG/Agent 安全评测工具

- 构建面向 LLM 应用的安全评测原型，覆盖 Prompt Injection、Jailbreak、敏感信息泄露、RAG 知识库投毒、Agent 工具越权等风险类型。
- 设计结构化测试样例与风险评分逻辑，对输入文本、检索片段、模型输出和工具调用参数进行统一检测，输出风险等级、命中证据和修复建议。
- 总结输入隔离、检索权限过滤、工具调用白名单、敏感信息脱敏和审计日志等防护策略，形成可复用 README 与测试用例。

## AI 辅助 SOC 告警研判与分级原型

- 构建 SOC 告警自动分级脚本，模拟处理 Web 攻击、异常登录、反弹连接、敏感文件访问等日志事件，并按资产重要性、攻击阶段和 IOC 特征计算优先级。
- 将告警映射到常见攻防语义标签与 MITRE ATT&CK 风格战术，生成面向安全分析师的摘要、处置建议和待验证证据。
- 通过样例 JSONL 数据与单元测试验证误报过滤、严重度排序和处置建议生成逻辑，为护网值守场景沉淀自动化分析能力。
