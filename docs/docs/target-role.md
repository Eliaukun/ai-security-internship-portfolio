## Typical Requirements

- Computer science, information security, or AI background.
- Red team, web security, vulnerability research, or blue team monitoring experience.
- Understanding of LLM application risks: prompt injection, jailbreak, sensitive data exposure, RAG poisoning, tool misuse, and Agent over-permission.
- Ability to write Python tools for evaluation, automation, log analysis, or security data processing.
- Good documentation habits and ability to turn experiments into reusable reports or tooling.
